### HOW TO RUN
```
    ### Auth Service
    $ cd apps/auth
    $ npm install
    $ npm run start:dev

    ### API Gateway
    $ cd apps/gateway
    $ npm install
    $ npm run start:dev

    ### Swagger
    http://localhost:2222/api/v1/swagger

```
Lưu ý: Mỗi service sẽ có file .env riêng => tạo .env theo .evv.example
